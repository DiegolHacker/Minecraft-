Preguntas

¿Cuál es la diferencia entre Internet y la World Wide Web?

El internet es una red de computadoras conectadas que transmiten información entre sí, la WWW es una serie de páginas web que existen encima del internet.

¿Cuáles son las partes de un URL?

Protocolo, subdominio, dominio, TLD, carpeta, página, etiqueta.

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

Get: Recibir información del servidor.

Head: Recibir solo el encabezado de la página.

Post: Subir información al servidor.

Put: Actualizar un dato del servidor o crear un dato nuevo.

Patch: Actualizar un dato de forma parcial.

Delete: Eliminar información.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

El método post ya que este método cifra en automático los datos por lo que son seguros y tiene una robustes y capacidad suficiente para enviar toda la informaicón.

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

Utiliza el método Get.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

Que todo salió bien, el servidor recibió la solicitud y la proceso de manera correcta.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

Sí debido a que es un error que aparece cuando un contenido no existe, y si un usuario común lo recibe es debido a una url desactualizada que lleva a un contenido eliminado.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

Sí debido a que es un error interno del servidor, y aunque el contenido existe y es accesible por el usuario no puede ser mostrado.

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

Que se encuentra en desuso debido a que existen métodos mejores para hacer la misma función, y se recomienda evitarlos ya que eventualmente pueden dejar de funcionar. el método font y center se encuentran obsoletos ya que se recomienda dar ese tipo de formato al texto a través de CSS.

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

HTML5 cuenta con métodos que dan un formato más entendible, como header, article, etc. También cuenta con mejores formularios con espacios dedicados a cada tipo de dato, como email que produce un cuadro de entrada sólo para correos electrónicos. La compatibilidad con celulares es mucho más amplia, y cuenta con tecnologías de forma nativa como reconocimiento de voz y contenido multimedia a más de 60 fps.

¿Qué componentes de estructura y estilo tiene una tabla?

De estructura existen th (Table Head), tr (Table Row), y td (Table data) que permiten llenar las celdas de una tabla con información, y de estilo thead, tbody, tfooter que determinan en que zona de la tabla se encuentran las celdas.

¿Cuáles son los principales controles de una forma HTML5?

Input, Fieldset, Output, Submit, Radio, Checkbox, Drop down list, Color, Date.
